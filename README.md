- React (v18.3.1)

- React hook form (v7.54.2)

- MUI (v6.4.2)

## Getting Started

### 1. Clone the repository:

```markdown
git clone https://github.com/your-username/project-name.git
cd project-name
```

### 2. Install dependencies

```markdown
npm install
# or
yarn install
```

### 3. Run the project locally:

```markdown
npm start
# or
yarn start
```
